# 2.1 Calculations

## 2.2 Feedback divider

The datasheet specifies that Vout in most cases, should not exceed 7V.

To set the desired output voltage of 5V, use the formula below, and tweak R1 and R2 values until, an acceptable balance between how accurate Vout is and using standard resistor values.

$R1 = R2\cdot \left(\dfrac{Vout}{0.6V}-1\right)→16.5kΩ\cdot\left(\dfrac{5V}{0.6V}-1\right)=121kΩ$

As these resistors are already readily available values, there is no need to adjust the values.

## 2.3 Input capacitance

For the input capacitor, the datasheet specifies that a capacitance of 22µF or more, generally is sufficient for most use cases.
