# 2.1 Calculations

## 2.2 Feedback divider

To calculate the resistances needed in the feedback divider, the formula below is used.

$R2 = \dfrac{0.8V}{Vout-0.8V}\cdot R1$

When calculating with R1 = 180kΩ, we get a resistance of 57.6kΩ for R2.

$R2 = \dfrac{0.8V}{3.3V-0.8V}\cdot 180kΩ = 57.6kΩ$

## 2.3 Input capacitance

For the input capacitor, the datasheet specifies that an X5R capacitor with a capacitance of 4.7µF or more is sufficient.

## 2.4 Output inductor

To choose a fitting output inductor, two parameters must first be calculated: Inductance and saturation current. 

The inductance is calculated using the formula below. It is suggested that the ripple current is assumed to be 40%.

$L = \dfrac {Vout\cdot \left(1-\frac {Vout}{Vin_{max}}\right)}{Fsw\cdot Iout_{max}\cdot 0.4}$

Calculating this, we get an inductance of 3.56µH. Fitting to a standard value, it is chosen to be 4.7µH. 

$L = \dfrac {3.3V\cdot \left(1-\frac {3.3V}{24V}\right)}{2MHz\cdot 1A\cdot 0.4} = 3.557µH$

Then saturation current is calculated. This is done using the formula below.

$Isat_{min} > Iout_{max} + \dfrac {Vout\cdot \left(1- \frac {Vout}{Vin_{max}}\right)}{2\cdot Fsw\cdot L}$

Calculating this, we get a minimum saturation current of 1.2A. This leaves us with the two necessary criterias to find a suitable inductor.

$Isat_{min} > 1A + \dfrac {3.3V\cdot \left(1- \frac {3.3V}{24V}\right)}{2\cdot 2MHz\cdot 4.7µH} = 1.2A$

A suitable inductor is found.
